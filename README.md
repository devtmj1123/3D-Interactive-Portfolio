# 🌌 Galaxy Portfolio

A stunning 3D interactive portfolio with galaxy theme, neumorphic design, and immersive Three.js experience.

## ✨ Features

- 🌌 **Galaxy Background** — Animated star field with parallax scrolling
- 🎨 **Neumorphic UI** — Modern design with subtle depth and glow effects
- 🚀 **3D Interactive Scene** — Three.js powered project showcase
- 📱 **Fully Responsive** — Works on all devices
- ♿ **Accessible** — WCAG 2.1 compliant
- ⚡ **Performance Optimized** — Lazy loading, device detection
- 🎭 **Smooth Animations** — Framer Motion powered transitions

## 🚀 Quick Start

```bash
# Clone the repository
git clone https://github.com/yourusername/galaxy-portfolio.git

# Navigate to project
cd galaxy-portfolio

# Install dependencies
npm install

# Start development server
npm run dev
```

## 📦 Tech Stack

| Technology | Purpose |

## 📁 Project Structure

```
src/
├── components/
│   ├── Background/      # Galaxy background effects
│   ├── Layout/          # Section components
│   ├── Scene/           # Three.js 3D scene
│   └── UI/              # Neumorphic components
├── styles/
│   ├── global.css       # Design tokens
│   └── neumorphic.css   # Component styles
├── utils/
│   ├── performance.js   # Optimizations
│   └── accessibility.js # A11y utilities
├── App.jsx              # Main app
└── main.jsx             # Entry point
```
